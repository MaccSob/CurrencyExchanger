# 💱 CurrencyExchanger

> A lightweight, user-friendly currency converter application that fetches real-time exchange rates to provide accurate financial calculations.

## 📌 About The Project
CurrencyExchanger is a functional application designed to simplify currency conversion for everyday users. The project showcases solid foundations in frontend layout design, user input handling, and interacting with currency data. It was built to practice clean code structure, state management, and creating responsive user interfaces.

## 🛠️ Tech Stack
*   **Core:** HTML5, CSS3, Django(Python)
*   **Styling:** Modern CSS (Flexbox / CSS Grid, Responsive Web Design)
*   **Data Source:** Integration with external exchange rates API

## 🌟 Key Features
*   📊 **Real-Time Conversion:** Instantly calculates values between multiple global currencies.
*   🔄 **Live Exchange Rates:** Fetches up-to-date financial data from a reliable public API.
*   📱 **Fully Responsive (RWD):** Smoothly adapts to all screen sizes, from mobile phones to large desktop monitors.
*   ⚡ **Dynamic UI Updates:** Immediate results without reloading the page, delivering a seamless user experience.
